## 四种引用
1. 强引用
	* 程序创建一个对象，并把这个对象赋给一个引用变量。被引用的Java对象绝不会被垃圾回收机制回收
2. 软引用
	* 系统内存足够时，不会被吸收；系统内存不足时，被回收。 通过SoftReference被实现
3. 弱引用
	* 当垃圾回收机制运行时，不管系统内存是否足够，都会被吸收。 通过WeakReference类实现
4. 虚引用
	* 不能单独使用。主要用于跟踪对象被垃圾回收的状态，必须和引用队列（ReferenceQueue）联合使用
